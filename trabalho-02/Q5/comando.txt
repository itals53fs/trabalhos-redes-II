Comando dig – Este comando permite fazer a consultas detalhadas ao serviço DNS mostrando o resultado. Para instalação no Debian  e Ubuntu  utilize o comando : apt-get install dnsutils
Utilizando o comando dig -v você verifica a versão garantindo que a instalação tenha sido concluída; Sua sintaxe simples será : dig [server] [name] [type]
Exemplos de utilização: Nome de Domínio : dig hostinger.com.
